# ATM Machine Project (Java)

## Project Description

This is a simple ATM Machine Mini Project developed using Java.  
The project simulates basic ATM operations like balance checking, deposit, withdrawal, and mini statement generation.

This project is created for mini project submission, Java practice, and placement preparation.

---

## Features

- ATM Number and PIN Validation
- View Available Balance
- Withdraw Money
- Deposit Money
- Mini Statement
- Change PIN Feature
- Exit Option
- Input Validation
- Method-Based Professional Structure

---

## Technologies Used

- Java
- Scanner Class
- Conditional Statements
- Loops
- Switch Case
- Basic OOP Concepts

---

## How It Works

1. User enters ATM Number
2. User enters ATM PIN
3. System validates login
4. ATM menu is displayed
5. User can perform:
   - Balance Check
   - Withdraw
   - Deposit
   - View Mini Statement
6. User exits safely
