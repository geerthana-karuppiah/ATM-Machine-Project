# ATM Machine Project (Java)

## Project Description

This is a simple ATM Machine Mini Project developed using Java.  
The project simulates basic ATM operations like balance checking, deposit, withdrawal, and mini statement generation.

This project is created for mini project submission, Java practice, and placement preparation.

---

## Features

- ATM Number and PIN Validation
- Check Available Balance
- Deposit Money
- Withdraw Money
- Transaction History using ArrayList
- Secure PIN Change
- Constructor-Based Initialization
- Input Validation
- Menu-Driven Interface
- Object-Oriented Programming (OOP)

---

## Technologies Used

- Java
- OOP (Classes and Objects)
- Encapsulation
- Constructors
- ArrayList
- Scanner Class
- Loops
- Switch Case
- Conditional Statements

---

## How It Works

1. User enters ATM Number and PIN
2. System validates credentials
3. ATM menu is displayed
4. User can:
   - Check Balance
   - Deposit Money
   - Withdraw Money
   - View Transaction History
   - Change PIN Securely
5. All transactions are stored dynamically using ArrayList
6. User exits the application safely
