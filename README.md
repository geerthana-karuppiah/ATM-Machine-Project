# ATM Machine Project – Java

A console-based ATM Machine project developed using Java.
This project was built step by step to learn Java concepts through real project development instead of only solving theory problems.
The project evolved version by version and each version introduced new programming concepts and software design improvements.

---

#  Project Objective

To simulate ATM operations and improve understanding of:

- Java Fundamentals
- Methods
- Object-Oriented Programming (OOP)
- Collections
- Exception Handling
- File Handling
- Software Design

---

- Java

#  Version History

#  Version 1 — Basic ATM System

## Features
- User Login
- Balance Check
- Deposit
- Withdraw
- Menu Driven Program

## Concepts Learned
- Variables
- Data Types
- Scanner
- If-Else
- Loops
- Methods

## Example Flow

Login
 ↓
Select Option
 ↓
Deposit / Withdraw
 ↓
Updated Balance

# Version 2 — ATM Using Methods

## Features
- Divided code into reusable methods
- Improved readability

## Concepts Learned
- Method Creation
- Method Calling
- Return Type
- Program Structure

### Benefit

Instead of writing everything inside `main()`,
operations are separated and easier to maintain.
